🔥 “LLM-Based Decision Justification System (Explainable AI Assistant)”
📌 Problem Statement
Build a system where:

* User asks a question requiring a decision (e.g., “Should I invest in X?”, “Is this resume good?”, “Which laptop is better?”)
The system:

1. Generates an answer
2. Explains WHY the answer is correct
3. Shows:
   * reasoning steps
   * assumptions
   * confidence level
4. Optionally:
   * shows alternative viewpoints
